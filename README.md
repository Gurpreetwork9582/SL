📚 Quiz Builder & Player (Streamlit App)

This application allows users to create and play quizzes easily using Excel/CSV files.

The app contains two main features:

1️⃣ Make Quiz
Users can download a pre-formatted Excel/CSV template containing the required columns:

Question

Option A

Option B

Option C

Option D

Answer

After downloading the template, users can fill in their own quiz questions, options, and the correct answer.

2️⃣ Quiz Play
Users can upload their completed Excel/CSV file, and the app will automatically generate an interactive quiz. Each question is displayed with four options, and when the correct answer is selected, the user can proceed to the next question.

This project is built using Python and Streamlit and demonstrates how structured data can be used to dynamically generate an interactive quiz experience.
